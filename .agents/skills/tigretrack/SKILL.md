# TigreTrack

Sistema institucional para la gestión, aprobación y seguimiento de solicitudes de cobertura de eventos para el área de Comunicación y Marketing Digital de UMAD e IMM.

## Stack Tecnológico

| Capa | Tecnología |
|---|---|
| Frontend | React, TypeScript, Vite, Axios |
| Backend | Node.js, Express, TypeScript |
| Base de datos | PostgreSQL, Prisma ORM |
| Servicios externos | Google Calendar API, Gmail API |
